# QA Portfolio – Mariana Santana de Soouza Soares

Este repositório contém projetos práticos desenvolvidos durante minha formação em Quality Assurance.

O objetivo é aplicar conceitos teóricos na prática, documentando todo o ciclo de testes de forma estruturada e profissional.

---

## Projeto 1 – Validação de Login (Guru99 Newtours)

Neste projeto foram realizados:

- Elaboração de Test Plan
- Criação de Casos de Teste manuais
- Escrita de cenários em BDD
- Execução de testes positivos e negativos
- Registro e classificação de bugs por severidade
- Análise de impacto no negócio
- Test Summary Report

---

## Estrutura do Repositório

- **TestPlan/** → Planejamento e relatório final de testes
- **TestCases/** → Casos de teste manuais e cenários em BDD
- **BugReports/** → Registro estruturado de defeitos encontrados

---

## Objetivo Profissional

Estou construindo este portfólio como parte da minha transição para a área de Tecnologia, com foco em Quality Assurance, desenvolvendo raciocínio analítico, organização e visão de qualidade desde o início.
